A*final.py is the python code for using A* to find a path in the maze

A*vis.py is the python code for seeing how A* work in the maze enviroment

buildmaze.py is for building a solvable maze and save map as a black block and white path(and build up maze model sdf) 

maze_world.sdf is the world file have maze in it

maze_model.sdf is the sdf file of the maze

coordinates_255.csv is for saving the point coordinates it need to slove the maze

runslovemaze.py is for generate different maze and slove it for ten times(run buildmaze->A* ten times), but it only save the last sdf file for the maze
